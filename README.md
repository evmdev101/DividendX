# DividendX Site

Static DividendX website for PulseChain.

## Local Preview

Run a simple static server from this folder:

```powershell
python -m http.server 8000
```

Then open:

```text
http://127.0.0.1:8000/index.html
```

## Pages

- Home
- Farms
- Dashboard

## GitHub Pages
